# :runner: ui-runner-vite

Simple ui runner using [vite](https://vitejs.dev/), configured for:

- [react](https://react.dev/)
- [typescript](https://www.typescriptlang.org/)

### Commands

```sh
# Install dependencies
$ nvm use
$ npm i

# Run UI
# http://localhost:5173/
$ npm run dev

# Run linter
$ npm run lint
```
